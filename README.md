# Web Patterns Playground

This repo explores common web and web-app UI patterns through lightweight, framework-free code (HTML, CSS, and a dash of vanilla JS). The goal is to show multiple ways to build familiar interface elements so designers can see how structure, styling, and interaction come together in code. It is an educational sandbox: by pairing simple source files with Figma prototypes, students connect the dots between what they draw and the materials that make digital products real. As your lecturer, I come from a classic design background, but I learned early that there is no true "no-code"—code is our material just as wood, metal, leather, or fabric are in physical design. Understanding that material, even if you later use no-code tools, makes you a stronger UI/UX designer.

Dieses Repository untersucht gängige Web- und Web-App-Patterns mit schlankem, frameworkfreiem Code (HTML, CSS und etwas Vanilla-JS). Ziel ist es, verschiedene Wege zu zeigen, wie bekannte UI-Elemente gebaut werden, damit Gestalterinnen und Gestalter erkennen, wie Struktur, Stil und Interaktion im Code zusammenspielen. Es ist eine Lernumgebung: Durch das Zusammenspiel einfacher Quelldateien und Figma-Prototypen entsteht ein klares Verständnis dafür, wie sich Entwürfe in reale digitale Produkte übersetzen. Als Dozent komme ich aus einer klassischen Designschule, aber ich habe früh verstanden, dass es kein echtes „No-Code“ gibt – Code ist unser Material, so wie Holz, Metall, Leder oder Stoff im Produktdesign. Wer dieses Material kennt, wird auch mit No-Code-Tools zu einer besseren UI/UX-Designer:in.
